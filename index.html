Bootcamp 2021: Snake Game

This is a platform game that I developed as part of the teaching the Programming in Java module to first year Computer Science students at City, University of London.

The game has the main character, a penguin with a helmet, which also know some magic too. It can be moved and interacted with via the keyboard, like pressing 'k' can do some magic stuff and attack enemies. The enemies will sensor the hero when we are close to it and start shooting the hero. We will get score every time the enemies been shooted. And sure the score is relavant to the time, the less time you use, the more points you get.

The game features a background soundtrack and sounds effect for different game effect, like shooting or pickups. And the bckground changes for different levels.Hope you like the game!
